Apa Itu Lokstra?

Lokstra adalah framework backend berbasis Go yang dirancang untuk membangun aplikasi backend secara sederhana, terstruktur, dan mudah dikembangkan—baik dalam bentuk monolith maupun microservice.

Dengan Lokstra, Anda dapat dengan mudah membuat service yang modular, terkonfigurasi melalui YAML, dan extensible menggunakan plugin .so. Menjadikannya ideal untuk proyek skala kecil hinga besar.

Lokstra juga menyediakan dukungan untuk multi-app, multi-port, dan multi-instance service dalam satu runtime, sehingga sangat fleksibel untuk berbagai kebutuhan pengembangan.

🚀 Simple

Minim boilerplate code, syntax yang intuitif, dan dokumentasi yang jelas. Lokstra memungkinkan developer untuk fokus pada logic bisnis tanpa terjebak dalam kompleksitas konfigurasi yang berlebihan.

⚙️ Scalable

Dukungan multi-app, multi-port, dan multi-instance service dalam satu runtime. Arsitektur yang fleksibel memungkinkan aplikasi tumbuh dari monolith ke microservice tanpa perlu refactoring besar-besaran.

🧱 Structured

Arsitektur modular dengan separation of concerns yang jelas. Konfigurasi terpusat melalui YAML dan sistem plugin .so memungkinkan pengembangan yang terorganisir dan mudah di-maintain dalam jangka panjang.

Quick Start

Mulai membangun service backend dengan Lokstra hanya dalam 3 langkah sederhana:

📦 Langkah 1: Setup Project

go mod init myapp && go get github.com/primadi/lokstra

✍️ Langkah 2: Tulis Kode

Buat file main.go:

package main

import "github.com/primadi/lokstra"

func main() {
    ctx := lokstra.NewGlobalContext()
    app := lokstra.NewApp(ctx, "myapp", ":8080")

    app.GET("/hello", func(c *lokstra.Context) error {
        return ctx.Ok("Hello from Lokstra!")
    })

    app.Start()
}

🚀 Langkah 3: Jalankan Server

go run main.go

Server akan berjalan di http://localhost:8080
